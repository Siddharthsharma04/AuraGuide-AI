# AuraGuide AI 👁️🔊

**AuraGuide AI** is a cutting-edge web application designed to empower the visually impaired with proactive navigation and semantic scene awareness. This project was developed as a final year BTech project in Software Engineering.

## 🌟 Unique Features
- **Proactive Awareness:** Real-time obstacle detection (people, cars, hazards) using **TensorFlow.js (COCO-SSD)**.
- **Contextual Scene Understanding:** Ask "What do you see?" for a voice summary of the environment.
- **Haptic Feedback Engine:** Distinct vibration patterns for obstacles and navigation turns.
- **Emergency SOS:** Voice-activated emergency mode that broadcasts location and alerts.
- **Accessibility-First UI:** High-contrast, oversized interface designed for low-vision users.

## 🛠️ Tech Stack
- **Framework:** React 19 + TypeScript
- **AI Model:** TensorFlow.js (COCO-SSD)
- **APIs:** Web Speech API (TTS/STT), Geolocation API, Vibration API
- **Tooling:** Vite, Lucide Icons

## 🏁 Installation & Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/Siddharthsharma04/AuraGuide-AI.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
